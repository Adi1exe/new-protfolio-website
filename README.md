# Portfolio Website

A modern, responsive personal portfolio website showcasing my work in **full-stack development** and **applied machine learning**.  
Built with a focus on clean UI, performance, scalability, and real-world engineering practices.

---

## Features

- Modern, minimal UI with responsive design
- Light / Dark mode support
- Modular and reusable component architecture
- Optimized for performance and accessibility
- Clean project structure and maintainable codebase

---

## Tech Stack

**Frontend**
- Next.js (App Router)
- React
- TypeScript
- CSS / PostCSS

**Tooling**
- Node.js
- npm / bun
- ESLint
- Git & GitHub

---

## Project Structure

```
├── src/
│ ├── app/ # App router pages & layouts
│ ├── components/ # Reusable UI components
│ ├── hooks/ # Custom React hooks
│ ├── lib/ # Utilities & helpers
│ └── styles/ # Global styles
├── public/ # Static assets
├── package.json
├── tsconfig.json
└── README.md
```

## Getting Started

### Clone the repository
```
git clone https://github.com/Adi1exe/orchids-portfolio-website-remix.git
cd orchids-portfolio-website-remix
```
---
### Install Dependencies
```
npm install
# or
bun install
```
### Run the Development Server
```
npm run dev
# or
bun run dev
```
